# Drill and Blast Cycle Variations

This repository contains graphical diagrams illustrating three innovative variations of the traditional drill and blast mining cycle.

## Cycle Variations

### 1. **Reduced Re-entry Cycle** (variant-1-reduced-reentry.svg)
- **Innovation**: Removes ventilation stage by using hydrogen peroxide and battery-electric equipment
- **Phases**: Drilling → Charging → Blasting → Mucking → Haulage → Support
- **Equipment**: Drill rig, Charging truck, Blasting system, Battery-electric loader, Battery-electric haulage truck, Shotcrete system
- **Key Benefit**: Reduced re-entry time, faster cycle, improved safety

### 2. **Hybrid Excavation Cycle** (variant-2-hybrid-excavation.svg)
- **Innovation**: Combines drilling, charging, blasting, and mucking into one continuous step
- **Phases**: Integrated Drill-Charge-Blast-Muck cycle → Ventilation → Support
- **Equipment**: Hybrid drill-blast unit, Mechanical cutting tool, Electric loader, Haulage truck, Ventilation fan, Shotcrete system
- **Key Benefit**: Reduced cycle time, continuous operation, improved productivity

### 3. **Improved Drill and Blast** (variant-3-improved-drill-blast.svg)
- **Innovation**: Includes out-of-cycle shotcrete application and automation
- **Phases**: Drilling → Charging → Blasting → Ventilation → Mucking → Out-of-Cycle Shotcrete (Automated) → Support
- **Equipment**: Automated drill rig, Charging truck, Blasting system, Ventilation fan, Electric loader, Haulage truck, Automated shotcrete robot, Shotcrete pump
- **Key Benefit**: Enhanced safety through automated shotcrete, better ground control, optimized cycle
